## Email Integration

A user sends an email to the secretary's office. As the project runs in the background, YAAC detects the new mail as an attendance list by checking the subject of the mail which contains specific phrases like "Anwesenheitsliste", "Anwesenheit", "Liste". If one of these phrases has been found, YAAC checks if the mail has an image as attachment.
